

h1. Run assembly


Run build using following syntax:

bc.. 
mvn -s settings.xml clean package

p. If you don't want to polute your .m2/repository specify alternative localRepository:


bc. 	<localRepository>C:\Java\tycho-maven-repository</localRepository>
